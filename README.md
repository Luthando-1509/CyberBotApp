# CyberBotApp
Cybersecurity Awareness Chatbot

Project Description

This project is a dedicated initiative to empower South African citizens with the knowledge needed to navigate the digital world safely. As cyber threats like phishing and identity theft become more common, this chatbot serves as a friendly, interactive assistant that teaches users how to spot red flags and protect their personal information. By simulating real-life scenarios, the bot provides practical, easy-to-understand advice on password security, safe browsing, and recognizing suspicious links, helping to build a more cyber-resilient community. 

Current Functionalities
Personalized Voice Greeting: A custom-recorded voice message that welcomes the user upon launching the application.
ASCII Art Branding: A visually engaging "CyberBot" logo displayed as a header in the console.
User Interaction: The bot greets users by name and provides personalized responses throughout the conversation.
Cybersecurity Knowledge Base: Predefined responses to common questions about password safety, phishing, and safe browsing.
Input Validation: Graceful handling of empty or unrecognized inputs with helpful rephrasing prompts.
Enhanced Console UI: A structured interface using colored text, dividers, and a typewriter effect to simulate a real conversation.

Project Stage and Future Advancements

Current Stage: Part 1 - Foundational Command-Line Application.

Future Advancements:

Transitioning from a console interface to a Graphic User Interface (GUI) using WPF or WinForms.
Implementing advanced topic recognition for more complex user queries.
Adding interactive elements, such as a cybersecurity tip task list or a simple educational game.
Expanding the knowledge base to cover emerging cyber threats specific to the South African landscape.
